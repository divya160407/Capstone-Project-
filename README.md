# Capstone-Project

A complete full-stack web page for university event registration, built with basic web developing tools and languages and deployed on local host infrastructure.

## Overview

- A full-stack university event registration web application developed as part of an academic capstone requirement.
- Allows students and faculty to browse upcoming campus events and register through a simple form-based interface.
- Built using core web technologies including HTML, CSS and deployed locally using php as a database.
- Follows a standard client-server architecture where the frontend communicates with a backend API to handle data.
- Uses a lightweight MySQL database to store and retrieve event and registration information.
- Designed to run entirely on a local development machine with no need for external hosting or cloud services.
- Users can view available events, fill out a registration form with their basic details, and receive an on-screen confirmation upon submission.
- Includes basic client-side form validation to ensure accurate data entry before submission.
- The absence of user authentication and limited scalability are known limitations of the current version.
- Serves as a solid academic demonstration and a foundation that can be expanded with additional features in future.
